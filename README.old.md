# hello-react-front-end


This project is fetching a random greeting from the API and displays it on the page.

## Build With

- JS
- React
- Redux


## Installation
```
  $ git clone git@github.com:SimonGrchevski/hello-react-front-end.git

  $ cd hello-react-front-end

  $ npm install

  $ npm start
  
```
